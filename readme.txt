Необходимые аддоны
	1. @A3MP
	2. @3d_editor

Запуск:
	В ActionManu выбрать <Editor>


Локальный объект - объект добавляется только на клиенте (без участии сервера). Объект нельзя разрушить и взаимодействовать. 
	!!! Объекты с дверми - необходимо ставить глобальными

Глобальный объект - объект созданный на сервере без каких либо ограничений
	

Управление:
	Камера: 
		1. ASDW -  движение в плоскости
		2. QZ - движение по вертикале
		3. LBM - поворот камеры

	Созадние объекта: 
		1. Выбрать класс объекта в нижнем левом списке 
		2. Перетащить название на нужное место в пространстве
		3. Выделить объекты, которые должны быть глобальными (по умолчанию все локальные) и нажать на CTRL+G (цвет иконки поменяется на оранжевый)
		4. Выедлить обхекты, которыые должны быть помечены на карте и нажать на [ (добавиться красные рамка у иконки)
		5. Для сохранения нажать END (скрипт добавлющийобхекты скопируется в буффер обмена)
		6. Перейти в текстовый редактор и вставить текс из буффера обмена
		7. Сохранить файл в корень миссии (например: base.sqf)
		8. В init.sqf добавить вызов созданного скрипта 
				execVM "base.sqf";

	Манипуляция над объектами:
		1. 	SPACE 	- перемещение по плоскости
		2. 	LALT 	- перемещение по вертикале
		3. 	LSHIT 	- поворот вокруг своей оси или выбранной оси
		4. 	LCTRL	- выделение объектов
		5. 	R		- выбор оси для поворота объектов
		6. 	CTRL+C	- копирование объектов (1 объект - без смещения, >1 объекта смещение обхектов берется от позиции мышки)
		7. 	CTRL+V	- вставить скопированные обеъекта  
		8. 	L 		- поставть обхекты на землю
		9. 	CTRL+L 	- наколнить обхект по наклону поверхности
		10. DELETE	- удалить выделенные объекты
		11. G		- сделать объект локальным
		12. CTRL+G	- сделать объект глобальным
		13. [		- добавить маркер на объект
		14. ]		- удалить маркер на объект 


	Манипуляция юнитом:
		M - перемещение юнита 

	Схоранение:
		END - сохранение скрипта в буффер обмена 

	Цвета: 
		синий 		- локальный объект
		оранжевый 	- глобальный объект
		красный 	- маркер на объект 




!!! Пожеления по переименованию объектов принимаются только в текстовом файле в определенном формате !!! 

1. При создании объекта в буффер обмена копируется класс объекта
2. Вставить текст в текстовый редактор
3. Изменить нужное поле
	textPlural - тип обхекта
    displayName - имя объекта

-------------------------------------------------
class Microphone3_ep1 : Microphone1_ep1 {
    textPlural = "Вещи";
    displayName = "Микрофон AAN";
};
-------------------------------------------------
